# Timeline

本项目为IOS嵌入式开发课程的实践项目

本项目旨在设计一款高效的记录时间、规划时间的应用。

在该应用中可以方便的记录过去时间的花费，并且可以设定计划、执行计划，计划的执行过程也会被记录下来。应用还提供了待办、统计、分类管理等功能，方便对任务进行管理、规划。
